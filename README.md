# OmniSched | 全域灵动调度 🚀
OmniSched 是一款重构安卓调度逻辑的现代化 Android 底层优化模组。我们摒弃了传统模组中「写死核心数」与「绑定特定处理器」的过时做法，透过 **全域动态运算**，将旗舰级的效能与能耗比带给每一台设备。

## 💡 为什么选择 OmniSched？
市面上多数的调度模组往往针对特定机型（如纯 8 核或纯高通平台）进行硬编码。一旦刷入不同架构的设备（如联发科、猎户座，或未来的异构多核），轻则无效，重则导致系统卡死、发热耗电。
* OmniSched 的诞生就是为了解决这个痛点：**一次刷入，全平台自适应**。

## 🌟 核心黑科技
 - 🧠 智能 SoC 动态嗅探
   - 开机瞬间自动侦测底层硬体。若为高通 (Qualcomm) 设备，自动解锁专属 QTI 底层优化与 Adreno 最佳调度；若为通用平台，则无缝切换至泛用型 Mali 动态调度。
 - ⚙️ Cpuset 动态几何分配
   - 自动读取设备核心架构，智能划分「大、中、小」核梯度。让前台应用 (Top-app) 独占最强算力，后台进程严格锁死于省电小核。
 - 🎮 底层渲染深度劫持 (A12+ 专属)
   - 在 post-fs-data 阶段提前接管渲染属性。全域强制开启 Vulkan 引擎与 SkiaVK 渲染，大幅降低 GPU 负载。
 - ⚖️ Schedutil 负载感知
   - 强制统一 CPU 调度器为 schedutil，实现效能与省电的动态平衡。
 
## 📱 系统相容性
 * 系统版本： Android 12 及以上 (API 31+)。
 * Root 环境： 支援 Magisk (v23.0+) / KernelSU / APatch。

## 📥 安装与使用
 * 在 Release 页面下载最新版的 OmniSched-xxx.zip。
 * 透过 Magisk/KernelSU 选择「从本机安装」。
 * 观察安装介面的设备侦测提示，重启手机。
